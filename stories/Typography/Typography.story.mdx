## <a id="importing"></a>Importing and Usage


```js
import GlobalStyle from '@catho/quantum/GlobalStyle';
```

Usually, it is used before the application component

```js
ReactDOM
  .render(() => (
    <>
      <GlobalStyle />
      <App />
    </>
  ), document.getElementById('root'));
```

The `<GlobalStyle />` will add and set automatically all the base configuration for font and [Icons](?path=/docs/foundation-icons--page).

If you want to include one of those separately, you can import each as you'd like to:

```js
import Typography from '@catho/quantum/GlobalStyle/Typography';
```
